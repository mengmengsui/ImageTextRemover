# SelectionBoxDeletion
目前最新最全的单个图片处理版本，支持：
- 上传一张图片，在页面上展示图片
- 在图片上框选要去除的文字区域
- 删除框选文字的选择框
- 调用Lama大模型，将文字去除，并填充适当的颜色
- 预览处理后的图片
- 下载处理后的图片
- 清空所选区域
- 清空当前所有

# SelectionBoxOnly
SelectionBoxOnly的前序版本，仅能增加框选区域和删除全部框选区域，不能逐个删除
